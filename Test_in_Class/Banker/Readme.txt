关于数据输入：
	因为是通过循环直接读输入内容放进二维数组，所以直接输入数字，输一个数回车一次。
	示例：输入测试用例1的MAX时：
				5<cr>
				5<cr>
				9<cr>
				5<cr>
				...（以此类推，直接输数）
*这个应该是要优化一下的地方，毕竟输那么多<cr>，不过不懂怎么弄了，思路是有，就是搞一个缓冲数组，然后把输入内容先放进去，然后再读里面的数其它的不管，可能还有其他方法，C的就不太懂了。
